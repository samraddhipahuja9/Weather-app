# Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript. This application allows users to search for any city and view real-time weather information such as temperature, humidity, wind speed, and weather conditions using a Weather API.

## Features

* Search weather by city name
* Real-time weather updates
* Displays temperature, humidity, and wind speed
* Dynamic weather icons
* Clean and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

This project was created to practice frontend web development concepts such as API integration, asynchronous JavaScript, DOM manipulation, event handling, and responsive web design.
